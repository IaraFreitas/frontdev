Linguagens

Conheça algumas das linguagens que utilizamos.

HTML

HTML é a "estrutura" do seu site! Ele organiza o conteúdo em blocos (como textos, imagens, vídeos), criando a base de tudo que você vê na web. Pense nele como o esqueleto que dá forma ao seu site.

CSS

CSS é como o "estilo" do seu site! Ele transforma o HTML básico em algo bonito e organizado, controlando cores, fontes, layout e muito mais. Pense nele como o responsável por vestir seu site com o melhor visual.

JAVA SCRIPT

JavaScript é quem dá vida às páginas, tornando tudo interativo. Com ele, os botões funcionam, os formulários enviam dados, e as animações acontecem. É o que faz o site agir e reagir às ações do usuário!
